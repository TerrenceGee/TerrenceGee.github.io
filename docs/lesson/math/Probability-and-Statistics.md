# 概率论与数理统计

## 1.随机事件与概率

### 考纲要求1：样本空间、随机事件、事件关系和运算

知识点1 随机事件：随机试验->样本空间、样本点->随机事件、基本事件、必然事件、不可能事件
知识点2 事件的关系和运算：

关系：三种关系（包含、相等、互不相容）
运算：四种运算（和、积、差、逆）-> 运算律：交换律、结合律、吸收律、分配律、德·摩根律（对偶律）

### 考纲要求2：概率、条件概率及基本性质、古典概率和几何型概率、概率加法、减法、乘法、全概率公式和贝叶斯公式

知识点1 概率概念： 随机事件与 P(A)一一对应 -> P(A)满足条件可成为概率 -> 概率的性质：边界、大小、运算 -> 四种运算公式与事件的四种运算一一对应
知识点2 古典概型和几何概型：

古典：场景一次性奖品抽取 -> 特征 -> 公式及使用组合表达的公式
几何： 场景平面坐标系取点 -> 特征 -> 公式

知识点3 条件概率与乘法公式： 条件概率计算公式 -> 另一种求法改变样本空间 -> 概率的乘法公式
知识点4 全概率公式 -> 公式；贝叶斯公式 -> 意义：由结果反向推导概率 -> 公式 -> 先验概率、似然概率、边缘概率、后验概率

### 考纲要求3：事件独立性，运用事件独立性进行概率计算

知识点1：事件独立定义 -> 两事件独立的结论 -> 独立与互斥的关系 -> 两两独立与相互独立的区别
知识点2：独立重复试验与伯努利概型： 两个结果、每次试验相互独立、进行 n 次 -> 伯努利概率公式

## 2.一维随机变量及其分布

### 考纲要求1：随机变量、分布函数、随机变量相联系事件的概率

知识点1：随机变量： 定义

> 随机变量：是一个将样本空间（其中是事件）映射到实数集的函数。 样本空间中的元素可以不是数，但是映射成随机变量后是实数。

知识点2：随机变量 -> 分布函数 -> 分布函数的性质（边界单调不减右连续） -> 用分布函数计算概率

### 考纲要求2：离散型随机变量及其概率分布、0-1分布、二项分布、几何分布、超几何分布、泊松分布

知识点1： 离散型随机变量： 定义 -> 概率分布律 -> 分布律性质 -> 概率的计算 \
知识点2：常见的离散型随机变量： 0-1分布 -> 二项分布 -> 泊松分布 -> 几何分布 -> 超几何分布 -> 泊松定理（将二项分布和泊松分布联系起来）

### 考纲要求3：连续型随机变量及其概率密度

知识点1：连续型随机变量： 两个微积分中的结论 -> 定义和结论 -> 概率密度充要条件 -> 由分布函数求概率密度函数 -> 概率的计算（单点的概率、某区间的概率）-> 由概率密度函数求分布函数 \
知识点2：常见的连续型随机变量：均匀分布 -> 指数分布 -> 正态分布 \

### 考纲要求4：求随机变量函数的分布

知识点1：如果是离散型随机变量：合并满足条件的概率值 \
知识点2：是连续型随机变量： 如果函数的分布是离散型，求分布律 -> 函数连续型，求分布函数，再求概率密度 -> 既不是离散，也不连续，求分布函数，使用分布函数法
