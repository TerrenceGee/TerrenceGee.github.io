PCDN

整体流程​

组件：终端机器 + PCDN节点镜像（OS）+ 宽带 + 中心化管理平台

  1. 购置服务器，配置是有要求的，其中最重要的是使用固态硬盘，服务器一般是 Dell 或者 Huawei
  2. 服务器安装相应的 OS 产品，涉及到系统 U 盘制作和 Linux 安装系统
  3. 开机，把 OS 绑定到 PCDN 商家的中心管理平台
  4. 在管理平台中配置节点的网卡信息，以能够上网。一般需要一个管理网卡和多个数据网卡
  5. 管理平台上选择相应的业务，选择完成后，节点自动开始被使用者调度。后面的步骤就是全自动的。

细节内容​

利益链条上的人​

  • 运营商人员（有话语权），能够无视掉运营商内部的教育
  • 装维人员：合同工，负责实际安装宽带，装宽带对他们来说也有绩效的好处，唯一的问题就是接受运营商的教育，不敢接触业务

运营商网络的一些考量指标​

运营商宽带的速度单位是 bps，与常说的 MB/s，相差一个8的倍数；一条宽带是按速度收费的，一般营销包括用户关注的都是
下行带宽，即用户从互联网获取信息的速度。PCDN 更关注的是上行速度，因为需要经常向互联网中推送数据。

  • 一般下行速度1500对应200的上行速度，单位都是 bps，资费在150-200元/月，仅供参考。

实操中的注意事项​

运营商的网络到终端用户一般是
运营商在地方设立的 PON 口设备箱 -> 分光器 -> 光猫 -> 路由器

一般按月开通的套餐，一个账号可以开通一个主带宽和一个副带宽，副带宽带宽较小，但是不需要额外花钱。

上网方式，默认是路由模式，这种模式下光猫可以作为wifi信号源使用，无论连接wifi还是连接以太网接线，终端可以直接连通互联网 对于机器，需要服务器拨号，这种上网方式的好处是公网ip是固定的，PPPoe拨号上网，账号和密码由装维人员提供

安装系统​

装系统：一次性 U 盘介质启动、设置引导顺序 镜像下载与U盘引导盘制作：一个工具可以方便制作U 盘，rufus，其作用是将.iso 格式的镜像写入 U 盘中 BIOS与UEFI引导方式：UEFI 较 BIOS 更先进，注意 OS 镜像刷机说明选择即可 Dell服务器设置RAID：只有 HDD 的 SATA 硬盘需要设置 RAID，M2接口的固态硬盘是不需要的。插入硬盘(foreign) -> phyical disk(ready) -> virual disk(online) Dell机器调节风扇大小，及服务器声音大小：使用一个工具：dell BMC，调节参数需要查阅资料

两种终端设备​

小盒子业务的优点是完全toC，不需要跟商家打交道，缺点是容易被封禁（盒子硬件限制大，跑省内不挣钱，跑省外是红线） 大节点业务的优点是不需要跑省外，即使是跑省内也很赚钱，参考1G上行带宽70块一天

防止封禁的措施​

1.不要跑省内 2.一个pon口不开多个宽带，一个pon最多开两个主宽
