# 英语单词

## 欧路词典

平台支持：
Web 端：通过官网（需搭配浏览器插件或手动导入）。\
PC 端：有 Windows/macOS 客户端，可直接添加生词。\
手机端：iOS/Android 同步学习。

### 使用

如何添加生词：

PC 端：

用欧路词典查词 → 点击“加入生词本”。

或直接 导入 Excel/CSV/TXT 词表（格式：单词 + 释义）。

Web 端：

使用浏览器插件（如“欧路词典划词翻译”）一键添加生词。

或登录官网手动管理生词本（功能较客户端少）。

### 分析

优点：

支持多词典扩展（柯林斯、牛津等）。

生词本自动同步，复习模式科学。

缺点：Web 端功能有限，建议用 PC 客户端操作。
