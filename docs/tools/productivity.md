# 生产力工具

1. 任务管理中枢：Notion

角色：所有信息的集散中心

关键配置：

主页看板：集成「每日任务」「项目追踪」「灵感库」数据库

模板库：

- 周计划模板（结合 Eisenhower Matrix）
- 读书笔记模板（Zettelkasten 方法）

移动端快捷入口：固定常用页面到手机桌面

2. 时间追踪：Toggl Track + Clockify

分工策略：

- Toggl Track：记录深度工作时段（与项目/任务标签绑定）
- Clockify：监控日常琐事耗时（如邮件/会议）

数据看板：每周导出CSV，用Notion仪表盘可视化分析时间黑洞

3. 知识管理：Obsidian

与Notion的分工：

- Notion：存放结构化数据（项目文档/协作内容）
- Obsidian：管理非结构化思考（日记/灵感/读书笔记）

核心插件：

- Daily Notes：晨间日记+晚间复盘
- Graph View：可视化知识关联

4. 专注执行：Forest + Focus@Will

组合使用：

- Forest（25分钟专注种树，防止手机干扰）
- Focus@Will（神经科学背景音乐，提升专注力）

数据联动：Forest专注时长自动记录到Toggl Track
