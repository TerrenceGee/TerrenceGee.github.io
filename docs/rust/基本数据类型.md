# Rust基本数据类型

## 变量与常量

| 类型 | 关键字 | 特性 |
| --- | --- | --- |
| 常量 | const | 不可变，不可覆盖 |
| 变量 | let (mut) | 可变可不变，可被覆盖 |
| 静态变量 | static (mut) | 可变可不变，不可被覆盖 |

## 数据类型

| 类型 | 名称 | 关键字 |
| --- | --- | --- |
| 标量类型 | 整型 |i8, i16, i32, i64, u8, u16, u32, u64 |
| 标量类型 | 浮点数 |f32, f64 |
| 标量类型 | 布尔值 |bool |
| 标量类型 | 字符 |char |
| 复合类型| 元组 |tuple|
| 复合类型 | 数组 | array|
| 自定义类型| 结构体 | struct|
| 自定义类型 |  枚举| enum |

> Rust是天生Unicode的，char类型占用四个字节。
