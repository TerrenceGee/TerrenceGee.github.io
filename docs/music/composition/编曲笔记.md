# 编曲

## 调式

一个八度有十二个音,故有十二个大调和十二个小调

中国风流行音乐调式的做法:
流行器乐部分,仍然使用原来的七声调式演奏
中国民乐器部分,使用五声调式演奏

## 节拍

节拍:衡量音乐的长度和节奏的基本单位,以强弱关系按照一定规律循环进行

4/4拍:以四分音符为一拍,每小节四拍,强弱关系为:强-弱-次强-弱
2/4拍
3/4拍:强-弱-弱
6/8拍:强-弱-弱-次强-弱-弱

强:down beat,弱:up beat,并不是指音量,是指拍子的决定性

## 音符

音符:记录音乐中的音的符号,包括音量,音高,音长,音色
音长:相对音长,节拍与时值
时值:音符的长短,音符在曲目中的相对长度

四分音符:1/4个全音,时值是四分音符的一拍
不加任何符号的四分音符,称作基本音符
单纯音符:只加横线或者下划线
附点音符:加 · ,表示1.5倍,有附点二分音符,附点四分音符,附点八分音符

音乐的速度:用每分钟的拍子数来衡量,其单位是bpm,beats per minute
每拍的真实时长,是用60秒➗速度

音符的高低: 全全半全全全半
"#"表示升高半音,"B"表示降低半音,"还原记号",临时标记仅对该小节内被标记的音符及其后面的相同音高音符起作用,一个小节内不用重复标记,如果小节内想将临时标记复原,只能通过"还原记号"

八度标记:高八度在上方加"-",低八度在下方加"-"
主音区(八度)音符不需要加八度标记,而不在主音区的音需要使用八度升降标记

## 记谱法

走带控制器（在 FL Studio 中在哪里？）
节拍器功能
主要是记谱法在 DAW 上音轨中的标记和作法

切分音：在音轨中，在任意单纯音符区域中，某个不从区域起点开始的音，经过了区域的中心，这个音就是切分音。音符区域可以是全音符，二分音符，四分音符等等

某个区域内，出现最多的八度当做主音区，即主音区是一个相对的概念。
