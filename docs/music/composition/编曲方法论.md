# 编曲

编曲一事，也有内功和外功之分。外功就是编曲的套路、经验、工作站的使用。内功则是音乐的一些理念，如
和弦、曲调、和声等等。掌握外功可以从实践入手，随着经验的增加而成长，而内功是不能通过实践系统习得。要想练好内功，可能还需要系统的学习乐理和编曲知识。笔者由于自己是一个门外汉，且时间有限，只能从外功入手，若某日有所小成，再考虑修习内功罢。

三步曲

1. 学习工作站软件的用法
2. 学习一些曲子的谱子，总结一首曲子的通用元素
3. 自行扒谱，逐渐形成自己的风格

## FL Studio工作站

- 播放列表(PlayList)：这里是轨道(track)的集合，多个轨道、轨道组组成一个完整的编曲作品
- 鼓机：
- 钢琴窗：
- 浏览器：各种音源的浏览

FL Studio或者说其他工作站的内容应该都是一样的：\
一个曲子由多个track组成 \
一个track(组)由一条音轨音乐组成，单轨音乐由标准音源(音色)的编排(和弦、节奏、强度等)或者采样音乐的编排组成 \
track组是有规律的，比如低音base，鼓(节拍)等等，这个要从许多作品中进行总结

> 写歌靠灵感,编曲靠悟性,混音靠耐心,成名靠人品
