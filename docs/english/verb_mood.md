# 动词语气

- 陈述语气, 与疑问语气可归于一类
- 祈使语气 imperative mood，命令和请求
- 虚拟语气 subjunctive mood, 条件从句

## 虚拟语气

### 第一种用法：表"不可能"假设，与事实相反的假设

- 和现在事实相反: 条件句：过去式，主句用 would do；这里的be一律用were
- 和将来事实相反（或者可能性很小的事实）：从句 should do , 主句用 would do
- 和过去事实相反：从句用have done, 主句用 would have done

### 第二种用法：虚拟语气表达愿望,请求，建议，命令

- wish：从句用动词过去式，be一律用were
- 对现在的愿望：wish + 过去式
- 对过去的愿望：wish + had/would have
- 对将来的愿望：wish + would/could/might + 动词原形

- would rather
- demand, sugguest, insist, order... + (should) + 动词原形
