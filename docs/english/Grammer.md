# English Grammer

## 动作（动词）、五种基本类型

1. 可以独立完成的动作：主语+不及物动词
2. 有一个动作的承受者：主语+单及物动词+宾语
3. 有两个动作的承受者：主语+双及物动词+间接宾语+直接宾语
4. 只有一个动作的承受者，但是需要补充：主语+复杂及物动词+宾语+（宾语）补语
5. 非动作、连系动词、系动词：主语+连系动词+主语补语（表语）

## 谓语动词三大本领

1. 动作时间：现在、过去、将来、过去将来 (Tense)
2. 动作状态：一般、完成、进行、完成进行 (Aspect)
3. 动作假设、情感等，即动作语气         (Mood)

- 虚拟语气
- ...

动词需要借助助动词来完成它的这些本领
助动词同时又可能有实义动词的含义，容易混淆

## 非谓语动词

除了作谓语外，动词还可以通过改变形式充当句子的其他部分：

- 主语
- 宾语
- 宾语补语
- 主语补语
- 定语
- ...

其形式：

- 动词不定式
- 现在分词
- 动名词
- 过去分词

非谓语动词可以取代从句，从而简化句子
